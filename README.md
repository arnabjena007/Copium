# Ghost Dashboard

Hackathon-ready cloud cost demo with:

- A Streamlit dashboard with fake-but-convincing FinOps metrics, Plotly charts, a demo toggle, ESG math, and an Ollama-backed consultant fallback.
- A one-page Next.js landing page with a vortex-style background that links into the Streamlit app.

## Run the dashboard

Install Python plus the packages in `requirements.txt`, then run:

```powershell
streamlit run streamlit_app.py
```

## Run the landing page

```powershell
cd web
npm.cmd install
npm.cmd run dev
```
